**CS451 Winter 2019 Checkers Application Repository**

**Prerequisites**  
```
nodejs
npm
pip3
python3.6+
```
  
**Usage**  
  
Get server requirements  
```
pip3 install -r requirements.txt
```

Get client requirements  
```
npm install
```
  
Run Server:  
```
python3 main.py
```

Run Client 
```
node index.js
```
  
**Client Usage**  
  
Each player needs to open  
```  
https://127.0.0.1:8080/
```
