## Prerequisites

```
nodejs
npm
pip3
python3.6+
```

## Installation

### Server


```
pip3 install -r requirements.txt
cd Server
python3 main.py
```

### Client

```
cd Client
npm install
node index.js
```

### Usage

Each player needs to open

```
https://127.0.0.1:8080/
```
